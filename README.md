# Binary Trees Solutions

A collection of efficient Python solutions for core **Binary Tree** algorithms from Codewars and LeetCode.

---

### !! Solved Challenges

* **Binary Tree Traversal**
    * *Pre-order, In-order, and Post-order* implementations using recursion.
* **Sort Binary Tree by Levels**
    * Level-order traversal (BFS) using `collections.deque` for **$O(n)$ efficiency**.
* **Delete Node in a BST**
    * Advanced deletion logic handling nodes with 0, 1, or 2 children while maintaining **BST invariants**.
